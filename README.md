# V-CPU
Virtual CPU

## Instruction Set & Binary Encoding:

[ OPCODE (4 bits) ][ R1 (2 bits) ][ R2 (2 bits) ][ IMM / ADDR (8 bits) ]

R0 = 00
R1 = 01
R2 = 10
R3 = 11

## Example Program (Assembly → Binary):

LDI R0, 5
0001000000000101

LDI R1, 1
0001010000000001

loop:
SUB R0, R1
0011000100000000

OUT R0
0111000000000000

JZ done
0110000000000110

JMP loop
0101000000000010

done:
HLT
1111000000000000
